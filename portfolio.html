Chat App

Design: Taneli Manninen

"Chat App" is a React Native mobile app project, which was a part of my thesis work that finished in the spring of 2024. During the development, the work focused on the UI and usability aspect of the mobile application.

The architecture of the app was based on a few user paths and the UI was build by using mockups. Functionally Chat App was a messaging app. User could register a user account, make posts and comment other posts. The server was built with Apollo Server + GraphQL and MongoDB was used as a database.

Future Factory
